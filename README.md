## Getting Started

1. Clone the repository.
2. Create a `.env` file based on `.env.example` and insert your `BOT_TOKEN`.
3. Install dependencies:

```bash
pip install -r requirements.txt
```

4. Run the bot:

```bash
python bot.py
```
